# Habit Tracker

Modern ve sade bir alışkanlık takip uygulaması.  
Günlük alışkanlıklarını ekleyebilir, tamamlananları işaretleyebilir, streak takibi yapabilir ve son 7 günü görsel olarak takip edebilirsin.

## Live Demo

[Live Site](https://slovi3.github.io/habit-tracker/)

## Features

- Yeni alışkanlık ekleme
- Kategori seçme
- Günlük tamamlandı / geri al işlemi
- Streak takibi
- Günlük ilerleme yüzdesi
- Progress ring görünümü
- Son 7 gün takibi
- Alışkanlık silme
- Tüm kayıtları temizleme
- LocalStorage ile veriyi koruma
- Responsive modern arayüz

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage
